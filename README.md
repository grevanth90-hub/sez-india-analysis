# India SEZ Spatial DiD Analysis

This repository contains code and analytical workflows for a spatial Difference-in-Differences analysis examining the impact of India’s Special Economic Zones (SEZs) on regional economic activity and environmental externalities.

The project combines:
- district-level socioeconomic datasets
- nighttime lights data
- PM2.5 environmental indicators
- spatial econometric methods
- Difference-in-Differences estimation

The analysis focuses on identifying whether SEZ policies generate net economic coordination benefits or primarily displace economic activity across regions.

Tools used:
- Python
- Pandas
- NumPy
- Statsmodels
- GIS/spatial workflows
